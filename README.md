# Artist
Mimics David Shrigley's work "the Artist" 's behavior to draw a picture.

模拟大卫·史瑞格里(David Shrigley) 的作品《艺术家》的绘画过程。

该作品于2018.09.08~2018.11.14在上海当代博物馆一楼“乱了乱了(Lose Your Mind)”展览中展出。
